# COMPILE — DSL Compiler Studio
A web-based compiler for the Option-1 Pascal-like DSL.

## Setup & Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the server:
   python app.py

3. Open in browser:
   http://localhost:5050

## Features
- Live tokenization, parsing, TAC, and assembly output
- Error reporting with exact line numbers
- 6 built-in example programs
- Syntax-highlighted output for all 4 compiler phases
- Keyboard shortcut: Cmd/Ctrl + Enter to compile
